画廊 component

类名 GalleryComponent
所属模块 GalleryModule
类型 组件
selector gallery

示例

  • html
  • typescript

属性

属性名 类型 输入/输出类型 说明
data Input any[] 包含图片地址的数据。
dataProps Input string[] data元素访问图片地址的key。如data=[{a:{b:{c:图片地址}}}]，dataProps应为['a','b','c']
size Input string 组件尺寸。默认全屏，可选lg、md、sm、xs
title Input string 标题

方法

方法名 参数 参数类型 说明
open ...args 不定参数

打开。参数长度和顺序不定。传入的：

第一个Event参数（模版中$event）作为触发事件；

第一个字符串作为单张图片地址，第二个以及第二个之后的字符串作为标题title；

第一个数字类型的参数作为浏览多张图片时当前查看图片的index；

第一个数组作为data，第二个数组作为dataProps。
close 无 - 关闭。
