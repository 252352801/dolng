文件上传控件

如何使用

1.导入 UploaderModule模块；

2.需要 UploaderDirective、Uploader、UploadFile 配合使用；

3.在input标签上使用uploader（UploaderDirective）指令，并初始化（配置）uploader指向的Uploader实例；

4.Uploader实例的queue属性以队列形式保存用选择的文件初始化的UploadFile实例。

注：暂无配套UI

示例

选择文件 上传
  • {{file.fileName}}
    {{file.progress}}%
  • html
  • typescript
UploaderDirective
类名 UploaderDirective
所属模块 UploaderModule
类型 指令
selector uploader
属性

@Input() uploader: Uploader;//Uploader实例

@Input() compressScale: number;//压缩尺寸

@Input() compressQuality: number;//压缩质量
Uploader
类名 Uploader
所属模块 UploaderModule
类型 服务
属性

url: string;//上传地址（必须）

headers: any[];//上传头部

method: string;//请求方法,默认post

uploadType: number;//上传方式 0:流式上传（默认） 1:base64上传

queue: UploadFile[];//上传队列

isPreview: boolean;//是否预览,默认true

isCompress: boolean;//是否压缩,默认false

maxSize: number;//最大文件大小（单个文件）

maxLength: number;//最大数量

customData: any;//用于存储自定义数据

name: string;//上传时默认的name

isUploading: boolean;//是否在上传
方法
方法名 参数 说明
upload 无 上传队列（queue）里的文件
onSelect
参数名 类型 描述
fn Function fn方法的参数为选择的文件files
【事件】选择文件后
onQueue
参数名 类型 描述
fn Function fn方法的参数为一个uploadFile实例
【事件】单个文件入列
onQueueAll
参数名 类型 描述
fn Function fn方法的参数为一个队列里的UploadFile实例（UploadFile[]）
【事件】全部文件入列
onUpload
参数名 类型 描述
fn Function fn方法的参数为当前Uploader实例
【事件】上传
onProgress
参数名 类型 描述
fn Function fn方法的参数percent（百分比）,uploadFile（当前UploadFile）,uploader（当前Uploader）,index（当前文件在queue里的index）
【事件】上传进度
onSuccess
参数名 类型 描述
fn Function fn方法的参数uploadFile（当前UploadFile）,uploader（当前Uploader）,index（当前文件在queue里的index）
【事件】单个文件上传成功
onComplete
参数名 类型 描述
fn Function fn方法的参数uploader（当前Uploader）
【事件】上传成功完成
onError
参数名 类型 描述
fn Function fn方法的参数uploadFile（当前UploadFile）,uploader（当前Uploader）,index（当前文件在queue里的index）
【事件】单个文件上传失败
onOverSize
参数名 类型 描述
fn Function fn方法的参数fileSize（当前文件大小），maxSize（允许的最大文件大小），uploadFile（当前UploadFile实例）
【事件】单个文件超过大小
onOverLength
参数名 类型 描述
fn Function fn方法的参数length（选择的文件数量），maxLength（允许的最大数量）
【事件】超过允许的数量
UploadFile
类名 UploaderDirective
所属模块 UploaderModule
属性

file: File;//原始文件对象

name: string;//上传时name

fileName: string;//文件名

fileExtension: string;//文件后缀名

fileType: string;//文件类型

fileSize: number;//文件大小，单位byte

dataUrl: string;//文件的base64数据

compressedDataUrl: boolean;//压缩后的文件数据

compressed: string;//是否被压缩过

progress: number;//上传进度（0-100）

uploaded: boolean;//是否已上传

success: boolean;//是否上传成功

error: boolean;//是否上传错误

response: any;//上传返回结果

customData: any;//用于存储自定义数据

xhr: XMLHttpRequest;//上传文件的XMLHttpRequest对象

submitData: {name: string,value: any}[];//上传时候的数据
方法
方法名 参数 说明
setSuccess 无 设置上传成功
setError 无 设置上传失败
getFile
参数名 类型 描述
type（选填） number 0:Filed/Blob对象 1:Base64数据
获取(要提交)的文件
addSubmitData
参数名 类型 描述
name string 字段名
value any 字段名对应的值
【事件】全部文件入列
